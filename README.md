## 📌 Project Description
The **Shipping Analysis Dashboard** is an interactive Power BI solution designed to analyze shipping performance, delivery modes, and profitability across different regions and customer segments.

This dashboard helps businesses:
- Optimize shipping strategies 🚚  
- Analyze profit by shipping modes 💰  
- Identify high-performing regions 🌍  
- Track monthly order trends 📈  
- Improve logistics decision-making 📦  

It converts raw operational data into meaningful insights for **efficient supply chain management**.

## 🚀 Features
- 📅 Year-wise filtering (2016–2019)
- 🧑‍💼 Segment selection (Consumer, Corporate, Home Office)
- 🚚 Shipping mode analysis (First Class, Second Class, Standard Class, Same Day)
- 📈 Monthly quantity trends by category
- 📊 Profit analysis by ship mode and category
- 🌍 State-wise profit distribution
- 📋 Detailed customer-level data table

- 📊 **Power BI Desktop**

- 🥧 **Quantity by Ship Mode** (Pie Chart)

- 📉 **Profit by State and Ship Mode** (Bar Chart)

---

## 📈 Key Insights
- 🚚 Standard Class shipping is the most frequently used mode.
- 💰 Standard Class generates the highest profit overall.
- ⚡ Same Day shipping has lower usage but higher urgency.
- 🌍 Certain states contribute significantly to total profit.
- 📊 Profit varies across shipping modes and product categories.

## ⚙️ How to Use
1. Clone or download this repository.
2. Open the `.pbix` file in **Power BI Desktop**.
3. Use slicers (Year, Segment) to filter data.
4. Interact with visuals to explore shipping insights.
